Release
--------------------------------------------------------------------------------
To install the "atlas" application, mount the "AtlasInstaller.dmg" in the "atlas/of_osx/release" folder. Then drag and drop the "atlas" application to the shortcut icon to your local "Applications" folder.

Source Code
--------------------------------------------------------------------------------
In order to run the source code of "atlas", you need to have openFrameworks for Mac OS X (http://www.openframeworks.cc/) and Xcode v4.x installed. Clone "atlas" and copy the source code in the "atlas/of_osx/apps/" subfolder into the "apps/" folder of your openFrameworks installation. Then open the "Atlas.xcodeproj" file and run the application by clicking the "Build and Run" button.
